## 1. 产品概述

深海渔船捕捞作业管理系统是专为远洋渔业设计的客户端软件，帮助船长和船员高效管理出海作业全流程，提升捕捞效率，保障航行安全，优化渔获管理和成本核算。

- 主要用途：航次计划管理、捕捞作业记录、渔获登记追踪、油料补给管理、船员排班调度、安全应急处置、收支结算分析
- 目标用户：渔船船长、轮机长、渔捞长、船员及渔业公司管理人员
- 市场价值：实现渔船作业数字化管理，提升生产效率，降低运营风险，保障作业安全

## 2. 核心功能

### 2.1 用户角色

| 角色 | 权限说明 | 核心功能 |
|------|----------|----------|
| 船长 | 系统最高权限 | 航次审批、收支结算、应急预案、数据导出 |
| 渔捞长 | 捕捞作业管理 | 渔场记录、网次登记、渔获录入 |
| 轮机长 | 设备油料管理 | 油料消耗、补给登记、设备状态 |
| 船员 | 个人作业查询 | 值更查看、个人信息、安全培训 |

### 2.2 功能模块

1. **出海计划**：航次计划创建、审批、执行状态追踪
2. **渔场记录**：渔场坐标管理、渔区信息、海况天气记录
3. **捕捞作业**：网次作业登记、拖网时长、捕捞深度、作业时间
4. **渔获登记**：渔获种类、重量统计、保鲜冷冻舱位分配
5. **油料补给**：油料消耗记录、补给港口管理、成本核算
6. **船员管理**：船员信息、排班调度、值更安排
7. **安全应急**：海况天气监测、落水救生预案、安全检查

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深海蓝(#0A2463)，象征海洋与专业
- **辅助色**：海洋青(#3E92CC)、警示橙(#FF6B35)、安全绿(#44AF69)、警告黄(#F9C80E)
- **中性色**：深灰(#1A1A2E)、中灰(#4A4A6A)、浅灰(#E8E8F0)、白色(#FFFFFF)
- **按钮风格**：圆角矩形(8px)，带有微妙的渐变和阴影，点击有按压反馈
- **字体**：标题使用 "Noto Sans SC Bold"，正文使用 "Noto Sans SC Regular"，数字使用等宽字体
- **布局风格**：侧边导航 + 主内容区，卡片式模块布局，数据表格清晰易读
- **图标风格**：线性图标，海洋主题元素(船锚、渔网、鱼群、海浪)，统一24px尺寸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 总览仪表盘 | 数据总览 | 统计卡片、进度圆环、折线图表、数据表格、预警横幅 |
| 出海计划 | 航次管理 | 时间线视图、表单弹窗、状态标签、航线地图预览 |
| 渔场记录 | 渔场管理 | 地图组件、坐标输入框、海况卡片、历史数据折线图 |
| 捕捞作业 | 作业登记 | 网次列表卡片、作业表单、计时器组件、深度剖面图 |
| 渔获登记 | 渔获管理 | 分类卡片、重量输入、舱位示意图、品质星级评分 |
| 油料补给 | 油料管理 | 油量表、消耗趋势图、港口选择器、成本计算器 |
| 船员管理 | 人员管理 | 船员卡片、排班日历、值更表、技能雷达图 |
| 安全应急 | 安全管理 | 天气仪表盘、预案流程图、安全检查清单、应急通讯录 |
| 收支结算 | 财务管理 | 收支饼图、利润趋势表、分类明细、导出按钮组 |

### 4.3 响应式设计

- **桌面优先**：针对1920×1080及以上分辨率优化，侧边栏固定宽度280px
- **平板适配**：1024px以下转为顶部导航，卡片自适应两列布局
- **触控优化**：按钮最小尺寸48×48px，表格支持横向滚动，下拉菜单优化触控区域
- **离线支持**：关键数据本地缓存，网络恢复后自动同步

### 4.4 交互与动效

- **页面加载**：骨架屏占位，内容淡入动画(300ms ease-out)
- **导航切换**：侧边栏滑入滑出，内容区左右滑动过渡
- **数据更新**：数字滚动动画，图表渐入绘制
- **表单交互**：输入框焦点发光效果，错误时抖动提示
- **按钮反馈**：悬停时上浮2px+阴影加深，点击时缩放0.98
- **状态变化**：状态标签颜色渐变过渡，进度条平滑填充
